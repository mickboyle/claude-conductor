## Database & Storage

**Storage Type:** File-based (Markdown + JSON)

All state is stored in the `conductor/` directory:
- Specifications: `conductor/tracks/*/spec.md`
- Plans: `conductor/tracks/*/plan.md`
- Progress: `conductor/tracks/*/metadata.json`
- Configuration: `conductor/*.md`

## Infrastructure & Deployment

**Distribution:** Git clone + plugin install

```bash
git clone <repo>
claude /plugin install ./claude-conductor
```

No build step required - plugin is pure Markdown/JSON.
